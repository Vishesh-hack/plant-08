# 🌿 plant.08

**plant.08** is an open-source beginner-friendly web project designed to help users learn how to grow and take care of plants in a simple and guided way.

---

## 📌 Problem Statement

Many beginners want to grow plants but struggle with:
- Lack of clear daily care steps
- No organized way to track plant growth
- Uncertainty about watering and sunlight needs

plant.08 solves this by providing users with easy plant care guidance and simple growth tracking.

---

## 💡 What This Project Does

plant.08 allows users to:
- View basic plant care guidance
- Track daily progress of plant growth
- Learn useful plant care information

(Current version is a basic prototype. Features will expand during the hackathon.)

---

## 🛠️ Tech Stack

- **Frontend:** HTML, CSS  
- **Future (optional):** JavaScript, Python Flask
